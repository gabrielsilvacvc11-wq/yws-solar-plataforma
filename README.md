# YWS Solar - Plataforma Completa

Site oficial da YWS Solar com design moderno, 4 simuladores interativos (Consumo, Área, Equipamentos, Inspeção/Limpeza), histórico local, geração de PDF profissional, área administrativa para a equipe, recomendações inteligentes e integração WhatsApp.

## Funcionalidades
- Design responsivo com modo escuro/claro
- Seletor de quantidade de painéis em tempo real (1 a 1500)
- Simulador de Consumo com cálculos baseados em irradiação por cidade
- Simulador de Área Disponível com fatores de telhado
- Escolha de Equipamentos (570W-630W + 9 marcas de inversores)
- Formulário de Inspeção e Limpeza com cálculo de deslocamento
- Histórico de simulações salvo no navegador
- Geração de PDF profissional com dados
- Área Admin protegida com busca e exportação CSV
- CTAs fortes para WhatsApp

## Como usar
Abra o index.html em qualquer navegador ou faça deploy no Vercel / GitHub Pages.

## Deploy no Vercel
1. Importe este repositório no Vercel
2. Deploy automático
